# Bundled example fixtures: sources and licenses

All fixtures in this directory are generated by the canonical generators in
`pyvista_cad/examples/_generate.py`. They are not derived from any third-party
data and ship under this project's MIT license.

| File                 | Generator function             | License |
| -------------------- | ------------------------------ | ------- |
| `plate.dxf`          | `_generate_dxf_plate`          | MIT     |
| `cube.step`          | `_generate_step_cube`          | MIT     |
| `bracket.step`       | `_generate_bracket_step`       | MIT     |
| `drawing.dxf`        | `_generate_drawing_dxf`        | MIT     |
| `assembly.3mf`       | `_generate_assembly_3mf`       | MIT     |
| `small_building.ifc` | `_generate_small_building_ifc` | MIT     |

To regenerate every fixture from scratch:

```bash
uv run python -m pyvista_cad.examples._generate
```

Generators depend on the optional generator stack: `build123d` (STEP),
`ezdxf` (DXF), `lib3mf` (3MF), and `ifcopenshell` (IFC). End users do not
need these tools to load the committed fixtures.
